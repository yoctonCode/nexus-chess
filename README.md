# NexusChess
A hobby-project desktop chess repertoire builder for beginners and intermediates. Made using C++ and the ImGUI library.

> [!WARNING]
> :rotating_light: **This hobby-project is a work-in-progress.**
>  If you want some already established app check out _encroissant_.

## Introduction
NexusChess is a desktop applications that allows its users to find, create, and train chess opening repertoires. It features vast customization options, cross-compatibility with Lichess, as well as an efficient spaced repetition based learning system.

## Getting Started
To get started, if you are on Windows, you can find the latest executable file in the release tab (https://github.com/yoctonCode/nexus-chess/releases/latest). If you are using Linux you may build the program with CMake, and then compile it using the compiler of your choice. However, be warned that this project was only intended for Windows and compatibility with other operating systems may not be supported.
